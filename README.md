# Homebrew Tap for Trinity

Homebrew formulae for [Trinity](https://github.com/abilityai/trinity) tools.

## Install

```bash
brew tap abilityai/tap
brew install trinity-cli
```

Or as a one-liner:

```bash
brew install abilityai/tap/trinity-cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `trinity-cli` | CLI for the Trinity Autonomous Agent Orchestration Platform |
